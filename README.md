<summary><b>⚙️ One-time setup for the live widgets above</b></summary>
<br>

This README has two self-updating sections that need a small GitHub Action enabled once on your <code>yuktac1011/yuktac1011</code> profile repo — after that they refresh automatically, no manual edits ever again.

**1. Recent Contributions feed** — add `.github/workflows/activity.yml`:
```yaml
name: Update Recent Activity
on:
  schedule:
    - cron: '0 */6 * * *'
  workflow_dispatch:
jobs:
  build:
    runs-on: ubuntu-latest
    permissions:
      contents: write
    steps:
      - uses: jamesgeorge007/github-activity-readme@master
        env:
          GH_TOKEN: ${{ secrets.GITHUB_TOKEN }}
          MAX_LINES: 3
```

**2. Contribution Snake animation** — add `.github/workflows/snake.yml`:
```yaml
name: Generate Snake
on:
  schedule:
    - cron: '0 */6 * * *'
  workflow_dispatch:
  push:
    branches: [ main ]
jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: yuktac1011
          outputs: dist/github-contribution-grid-snake-dark.svg
      - uses: crazy-max/ghaction-github-pages@v4
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

Commit both files to your profile repo and the **Recent Contributions** list and **snake graphic** will start animating and updating on their own.
